# 认知障碍志愿者服务规范

**制定时间：** 2026-03-13 02:55  
**适用范围：** 所有服务认知障碍患者的志愿者  
**规范等级：** 强制性执行标准

## 🎯 服务宗旨与原则

### 核心宗旨
> **"专业、尊重、安全、共情"**

### 基本原则
1. **以人为本**
   - 尊重患者的个人尊严和自主权
   - 关注个体需求和偏好
   - 维护患者的隐私和 confidentiality

2. **安全第一**
   - 确保患者人身安全
   - 保障志愿者自身安全
   - 预防意外事件发生

3. **专业规范**
   - 遵循标准化服务流程
   - 使用科学有效的方法
   - 持续学习和提升

4. **团队协作**
   - 与家属保持良好沟通
   - 与专业团队密切配合
   - 志愿者之间相互支持

## 📋 志愿者资质要求

### 基本条件
1. **年龄要求**
   - 年满18周岁
   - 特殊情况需监护人同意

2. **健康状况**
   - 身体健康，无传染性疾病
   - 心理状态稳定，抗压能力强
   - 无影响服务的身体限制

3. **时间承诺**
   - 每周至少服务4小时
   - 连续服务不少于3个月
   - 按时参加培训和会议

### 培训要求
1. **必修培训（服务前完成）**
   - 认知障碍基础知识（8学时）
   - 安全照护技能（4学时）
   - 沟通技巧（4学时）
   - 应急处理（2学时）
   - 考核合格后方可上岗

2. **持续培训（服务期间）**
   - 每月至少参加1次专题培训
   - 每季度参加案例讨论会
   - 每年完成继续教育学时要求

### 考核认证
1. **上岗考核**
   - 理论知识考试（80分合格）
   - 技能操作考核（通过性）
   - 情景模拟测试（通过性）

2. **定期评估**
   - 每月服务表现评估
   - 每季度技能复训
   - 年度资格复审

## 🏠 服务环境规范

### 居家服务环境
1. **安全评估**
   - 首次服务前进行环境安全评估
   - 识别潜在风险因素
   - 提出改进建议

2. **环境准备**
   - 确保活动空间充足
   - 移除危险物品
   - 准备必要设备

3. **隐私保护**
   - 尊重家庭隐私
   - 不随意拍照录像
   - 保护患者个人信息

### 机构服务环境
1. **遵守机构规定**
   - 了解并遵守机构规章制度
   - 配合机构工作人员
   - 维护机构秩序

2. **环境适应**
   - 熟悉机构布局
   - 了解应急通道
   - 掌握设备使用方法

## 🤝 服务流程规范

### 服务前准备
```
1. 服务计划确认
   ├── 确认服务对象信息
   ├── 了解患者当前状况
   ├── 明确本次服务目标
   └── 准备所需物资

2. 个人准备
   ├── 穿着得体（统一服装）
   ├── 佩戴工作证件
   ├── 携带服务记录表
   └── 准备应急联系方式

3. 心理准备
   ├── 调整到服务状态
   ├── 复习相关知识和技能
   ├── 设定合理期望
   └── 做好应急预案
```

### 服务中执行
```
1. 到达与问候（0-5分钟）
   ├── 准时到达，提前通知
   ├── 礼貌问候患者和家属
   ├── 确认服务内容和时间
   └── 进行简短的环境评估

2. 服务实施（核心时段）
   ├── 按照计划开展活动
   ├── 观察患者反应和状态
   ├── 灵活调整活动内容
   └── 确保活动安全进行

3. 沟通与记录（贯穿全程）
   ├── 与患者保持良好沟通
   ├── 及时与家属交流情况
   ├── 记录重要观察和事件
   └── 拍照记录需获得同意

4. 结束与交接（最后10分钟）
   ├── 逐步结束活动
   ├── 协助整理环境
   ├── 向家属反馈情况
   └── 确认下次服务安排
```

### 服务后工作
```
1. 立即工作
   ├── 填写服务记录表
   ├── 记录重要观察发现
   ├── 标注需要关注的问题
   └── 整理使用过的物品

2. 后续跟进
   ├── 必要时电话随访
   ├── 与其他志愿者交流
   ├── 向督导汇报情况
   └── 准备下次服务计划
```

## 🗣️ 沟通规范

### 与患者沟通
1. **基本原则**
   - 保持尊重和耐心
   - 使用简单清晰的语言
   - 给予充分反应时间
   - 关注非语言沟通

2. **沟通技巧**
   - **称呼：** 使用患者喜欢的称呼方式
   - **语速：** 缓慢、清晰、平稳
   - **内容：** 一次只说一件事
   - **反馈：** 确认理解，重复关键信息

3. **特殊情境处理**
   - **重复提问：** 耐心回答，不表现出不耐烦
   - **错误认知：** 不争论，转移话题或验证感受
   - **情绪激动：** 保持冷静，安抚情绪

### 与家属沟通
1. **沟通时机**
   - 服务开始前：了解最新情况
   - 服务过程中：重要变化及时沟通
   - 服务结束后：全面反馈

2. **沟通内容**
   - 患者当天的表现和反应
   - 观察到的积极变化
   - 需要注意的问题
   - 建议和资源推荐

3. **沟通方式**
   - 尊重家属的照护经验
   - 提供专业建议但不强加
   - 保持适当的边界

### 团队内部沟通
1. **交接班沟通**
   - 患者最新状况
   - 需要注意的事项
   - 未完成的工作
   - 家属的特殊要求

2. **案例讨论**
   - 定期案例分享
   - 问题解决方案讨论
   - 经验教训总结

## 🛡️ 安全规范

### 患者安全
1. **人身安全**
   - 防止跌倒：全程陪同，环境安全
   - 防止走失：外出时密切看护
   - 防止误食：监督进食，管理药物

2. **心理安全**
   - 避免批评和指责
   - 保护自尊心
   - 提供情感支持

3. **医疗安全**
   - 不提供医疗建议
   - 不协助服药（除非经过专门培训）
   - 及时报告异常情况

### 志愿者安全
1. **人身安全**
   - 不单独进入陌生环境
   - 保持通讯畅通
   - 遇到危险及时求助

2. **健康安全**
   - 注意个人卫生
   - 做好感染防护
   - 关注自身心理健康

3. **法律安全**
   - 遵守相关法律法规
   - 明确服务边界
   - 购买必要保险

### 环境安全
1. **风险评估**
   - 服务前环境评估
   - 识别潜在危险
   - 采取预防措施

2. **应急准备**
   - 熟悉应急流程
   - 掌握急救技能
   - 准备应急物资

## 📝 记录与报告规范

### 服务记录要求
1. **记录内容**
   - 服务日期、时间、地点
   - 服务内容和活动
   - 患者反应和表现
   - 重要观察发现
   - 与家属沟通情况
   - 下次服务建议

2. **记录原则**
   - **及时性：** 服务结束后立即记录
   - **准确性：** 客观描述，避免主观判断
   - **完整性：** 重要信息不遗漏
   - **保密性：** 妥善保管，不泄露

### 报告要求
1. **日常报告**
   - 每次服务后提交服务记录
   - 每周提交服务总结
   - 每月参加服务汇报会

2. **特殊事件报告**
   - **立即报告：** 安全事故、医疗紧急情况
   - **24小时内报告：** 患者状况显著变化
   - **48小时内报告：** 服务中的问题和困难

3. **报告渠道**
   - 直接督导
   - 项目协调员
   - 紧急情况热线

## ⚠️ 禁止行为

### 绝对禁止行为
1. **身体接触**
   - 不当的身体接触
   - 未经同意的拥抱或抚摸
   - 任何形式的体罚

2. **言语行为**
   - 辱骂、嘲笑、贬低患者
   - 大声呵斥或威胁
   - 使用歧视性语言

3. **经济行为**
   - 向患者或家属借钱
   - 接受贵重礼物
   - 推销产品或服务

4. **其他行为**
   - 私自更改服务内容
   - 泄露患者隐私信息
   - 未经允许拍照录像

### 限制行为
1. **需要批准的行为**
   - 带患者外出活动
   - 更改服务时间
   - 接受媒体采访

2. **需要培训的行为**
   - 协助服药
   - 处理医疗紧急情况
   - 进行专业评估

## 🔄 质量改进

### 服务质量评估
1. **评估维度**
   - 服务规范性
   - 患者满意度
   - 家属满意度
   - 服务效果

2. **评估方法**
   - 服务记录检查
   - 现场观察
   - 满意度调查
   - 案例讨论

3. **评估频率**
   - 每月：服务记录检查
   - 每季度：满意度调查
   - 每半年：全面评估

### 持续改进机制
1. **问题反馈**
   - 建立问题反馈渠道
   - 鼓励志愿者提出建议
   - 及时处理投诉和建议

2. **培训更新**
   - 根据问题更新培训内容
   - 引入新的知识和技术
   - 定期复训重要技能

3. **规范修订**
   - 每年审查服务规范
   - 根据实践修订内容
   - 确保规范的适用性

## 🎖️ 志愿者权益保障

### 基本权益
1. **培训权利**
   - 获得必要的岗前培训
   - 参加持续教育培训
   - 获得学习资料和支持

2. **工作条件**
   - 安全的工作环境
   - 必要的工作设备
   - 合理的服务安排

3. **支持保障**
   - 督导指导和支持
   - 团队协作和互助
   - 心理支持和疏导

### 认可与激励
1. **服务认可**
   - 服务时间记录和认证
   - 服务表现评价和反馈
   - 优秀志愿者表彰

2. **成长机会**
   - 技能提升培训
   - 领导力发展机会
   - 职业发展指导

3. **福利待遇**
   - 服务期间保险
   - 交通和餐食补贴（如适用）
   - 参与活动的机会
