# 认知障碍日常护理规范

**制定时间：** 2026-03-13 03:00  
**适用对象：** 志愿者、家属、护理人员  
**护理级别：** 基础到中级护理技能

## 🏠 日常生活护理体系

### 护理理念
> **"维持功能，促进独立，确保安全，提升质量"**

### 护理目标层次
```
基础目标：确保基本生存需求
├── 安全的居住环境
├── 充足的营养摄入
├── 基本的个人卫生
└── 必要的医疗照护

中级目标：维持现有功能
├── 保持活动能力
├── 维持认知功能
├── 促进社交参与
└── 管理行为症状

高级目标：提升生活质量
├── 尊重个人尊严
├── 满足心理需求
├── 提供有意义的活动
└── 支持家庭照护者
```

## 🌅 晨间护理流程

### 起床协助（6:00-8:00）
```
1. 唤醒
   ├── 轻声呼唤，轻触手臂
   ├── 告知时间、天气、当日安排
   ├── 给予充分适应时间（5-10分钟）
   └── 观察精神状态和情绪

2. 如厕协助
   ├── 询问是否需要如厕
   ├── 必要时协助到卫生间
   ├── 确保安全（扶手、防滑）
   └── 尊重隐私，逐步退出

3. 个人卫生
   ├── 洗脸：温水，动作轻柔
   ├── 刷牙：协助或监督
   ├── 梳头：按个人习惯整理
   └── 剃须/化妆：按需协助

4. 穿衣协助
   ├── 准备简单易穿的衣服
   ├── 按穿着顺序摆放
   ├── 口头指导，必要时协助
   └── 检查是否舒适得体
```

### 早餐准备与进食（7:00-8:30）
```
1. 环境准备
   ├── 安静、明亮、整洁
   ├── 移除干扰物（电视、手机）
   ├── 准备舒适的座位
   └── 检查餐具安全性

2. 食物准备
   ├── 营养均衡：蛋白质+碳水化合物+蔬果
   ├── 易于咀嚼和吞咽
   ├── 温度适宜（测试手腕内侧）
   ├── 小份量，多餐次

3. 进食协助
   ├── 口头提示进食步骤
   ├── 示范动作，鼓励模仿
   ├── 必要时喂食（从侧面）
   ├── 给予充足时间（30-45分钟）

4. 进食后
   ├── 检查口腔清洁
   ├── 协助清洁面部和手部
   ├── 记录进食情况
   └── 安排适当休息
```

## ☀️ 日间活动安排

### 认知活动（9:00-11:00）
```
1. 记忆训练（20-30分钟）
   ├── 近期事件回忆（昨天做了什么）
   ├── 照片回忆（家人照片识别）
   ├── 物品记忆游戏（5个物品，回忆）
   └── 使用记忆辅助工具（日历、备忘录）

2. 定向训练（15-20分钟）
   ├── 时间定向：日期、星期、季节
   ├── 地点定向：房间功能、家庭布局
   ├── 人物定向：家人照片、称呼练习
   └── 使用定向辅助工具（大钟、日历）

3. 执行功能训练（20-30分钟）
   ├── 简单计划：制定购物清单
   ├── 分类活动：物品分类（衣物、食品）
   ├── 排序练习：事件顺序排列
   └── 问题解决：情景模拟处理
```

### 身体活动（11:00-12:00）
```
1. 热身运动（5分钟）
   ├── 颈部转动（缓慢）
   ├── 肩部环绕
   ├── 手腕脚踝转动
   └── 深呼吸练习

2. 有氧运动（15-20分钟）
   ├── 室内散步（来回走动）
   ├── 原地踏步
   ├── 简单舞蹈动作
   └── 根据能力调整强度

3. 平衡训练（10分钟）
   ├── 坐姿平衡（有支撑）
   ├── 站立平衡（扶墙或椅子）
   ├── 单腿站立（短暂，有保护）
   └── 走直线练习

4. 柔韧性训练（5分钟）
   ├── 上肢伸展
   ├── 下肢伸展
   ├── 躯干转动
   └── 注意安全，避免过度
```

### 午餐与午休（12:00-14:00）
```
1. 午餐准备（12:00-12:30）
   ├── 营养均衡：主食+蛋白质+蔬菜
   ├── 易于消化：软食、切碎
   ├── 注意饮水：保证充足液体摄入
   └── 营造愉快进餐氛围

2. 午休安排（13:00-14:00）
   ├── 创造安静环境
   ├── 调整光线和温度
   ├── 提供舒适的休息位置
   └── 午休时间30-60分钟为宜
```

```

   ├── 洗澡协助（根据需要）
   ├── 口腔清洁
```

```
1. 睡眠监测
   ├── 定期查看（每2-3小时）
   ├── 观察呼吸和姿势
   ├── 记录异常情况
   └── 必要时协助调整姿势

2. 如厕协助
   ├── 定时提醒（根据习惯）
   ├── 夜间使用便器或纸尿裤
   ├── 确保安全（照明、防滑）
   └── 快速完成，减少清醒时间

3. 异常处理
   ├── 夜间躁动：安抚，不开灯
   ├── 噩梦惊醒：陪伴， reassurance
   ├── 游走：温和引导回床
   └── 紧急情况：立即求助
```

## 🛁 个人卫生护理细则

### 洗澡护理
```
1. 准备工作
   ├── 室温：24-26°C
   ├── 水温：37-40°C（用手腕测试）
   ├── 准备：毛巾、浴袍、洗浴用品
   └── 安全：防滑垫、扶手、椅子

2. 洗澡过程
   ├── 尊重隐私，逐步暴露
   ├── 从头到脚，从前到后
   ├── 动作轻柔，观察反应
   └── 注意褶皱部位清洁

3. 特殊情况
   ├── 拒绝洗澡：找出原因，尝试不同时间
   ├── 攻击行为：暂停，安抚，寻求帮助
   ├── 皮肤问题：轻柔处理，咨询医生
   └── 身体限制：使用辅助设备
```

### 口腔护理
```
1. 刷牙协助
   ├── 使用软毛小头牙刷
   ├── 牙膏少量（豌豆大小）
   ├── 按顺序清洁（外、内、咬合面）
   └── 鼓励自己完成部分

2. 义齿护理
   ├── 每日清洗义齿
   ├── 夜间取下浸泡
   ├── 检查口腔黏膜
   └── 定期专业检查

3. 特殊问题
   ├── 牙龈出血：轻柔，咨询牙医
   ├── 口腔干燥：适量饮水，人工唾液
   ├── 吞咽困难：注意安全，防误吸
   └── 口腔感染：及时就医
```

### 皮肤护理
```
1. 日常护理
   ├── 每日全身检查（特别是骨突处）
   ├── 保持皮肤清洁干燥
   ├── 使用温和护肤品
   └── 避免过热或过冷

2. 压疮预防
   ├── 定期翻身（卧床者每2小时）
   ├── 使用减压垫
   ├── 保持床单平整
   └── 营养支持（足够蛋白质）

3. 皮肤问题处理
   ├── 红肿：减压，观察
   ├── 破损：清洁，保护，就医
   ├── 瘙痒：避免搔抓，咨询医生
   └── 皮疹：识别原因，对症处理
```

## 🍽️ 营养与饮食管理

### 营养需求评估
```
1. 基本需求
   ├── 热量：根据活动水平调整
   ├── 蛋白质：1.0-1.2g/kg体重
   ├── 水分：1500-2000ml/天
   └── 维生素矿物质：均衡饮食

2. 特殊需求
   ├── 吞咽困难：软食、糊状食物
   ├── 食欲下降：小份多餐，增加风味
   ├── 体重下降：高能量高蛋白补充
   └── 便秘：增加纤维和水分
```

### 进食安全
```
1. 姿势安全
   ├── 坐直，头微前倾
   ├── 下巴内收
   ├── 双脚平放地面
   └── 必要时使用特殊餐具

2. 食物安全
   ├── 避免粘性食物（年糕、汤圆）
   ├── 避免带骨、刺食物
   ├── 食物温度适宜
   └── 小块、易咀嚼

3. 观察要点
   ├── 咀嚼和吞咽能力
   ├── 进食速度和量
   ├── 呛咳迹象
   └── 进食后反应
```

### 饮水管理
```
1. 饮水计划
   ├── 晨起：200ml温水
   ├── 上午：300-400ml
   ├── 下午：300-400ml
   ├── 傍晚：200-300ml
   └── 睡前2小时：限制饮水

2. 饮水方式
   ├── 使用防洒杯子
   ├── 小口慢饮
   ├── 必要时使用吸管
   └── 记录出入量（必要时）
```

## 🚽 排泄护理

### 如厕训练
```
1. 定时如厕
   ├── 建立固定时间表
   ├── 餐后30分钟提醒
   ├── 睡前必须如厕
   └── 夜间定时提醒

2. 环境适应
   ├── 卫生间标识明显
   ├── 夜间照明充足
   ├── 坐便器高度合适
   └── 准备必要的辅助设备

3. 失禁处理
   ├── 保持冷静，不责备
   ├── 及时清洁，保持干燥
   ├── 使用合适的护理产品
   └── 记录失禁情况，寻找规律
```

### 便秘预防
```
1. 饮食调整
   ├── 增加膳食纤维（蔬菜、水果）
   ├── 保证充足饮水
   ├── 适量油脂摄入
   └── 避免过多精制食物

2. 活动促进
   ├── 规律身体活动
   ├── 腹部按摩（顺时针）
   ├── 建立排便习惯
   └── 必要时使用缓泻剂（遵医嘱）
```

## 🧘‍♀️ 行为症状管理

### 激越行为处理
```
1. 识别诱因
   ├── 身体不适（疼痛、饥饿、如厕）
   ├── 环境因素（噪音、人群、变化）
   ├── 心理因素（恐惧、困惑、无聊）
   └── 沟通障碍（不理解、表达困难）

2. 干预策略
   ├── 保持冷静，降低声调
   ├── 转移注意力（喜欢的事物）
   ├── 提供安全空间
   └── 必要时寻求帮助

3. 预防措施
   ├── 建立规律生活
   ├── 避免过度刺激
   ├── 提供有意义的活动
   └── 确保基本需求满足
```

### 游走行为管理
```
1. 安全措施
   ├── 佩戴身份标识
   ├── 安装门禁报警
   ├── 邻居和社区告知
   └── 准备近期照片

2. 活动安排
   ├── 提供安全游走空间
   ├── 安排规律散步时间
   ├── 提供室内活动替代
   └── 满足活动需求

3. 应急准备
   ├── 制定寻找计划
   ├── 准备应急联系人
   ├── 了解社区资源
   └── 定期演练
```
