# 知行中国：脑健康志愿者培训平台

**项目类型：** 社会企业项目（公益+商业双轮驱动）  
**启动时间：** 2026年3月  
**项目阶段：** MVP规划与准备  
**项目经理：** 全真  
**技术助手：** 小青

## 🎯 项目愿景

打造中国领先的脑健康志愿者培训与服务平台，通过标准化培训、能力认证和服务匹配，提升脑健康服务的可及性和专业性。

## 🛠️ 技术栈选择

### 移动端（React Native）：
- **框架：** React Native 0.75+
- **UI库：** React Native Paper / NativeBase
- **状态管理：** Redux Toolkit / Zustand
- **导航：** React Navigation 6.x

### Web端（React）：
- **框架：** React 18+
- **UI库：** Ant Design / Material-UI
- **状态管理：** Redux Toolkit
- **路由：** React Router 6.x

### 后端（Node.js）：
- **运行时：** Node.js 20+
- **框架：** Express.js / NestJS
- **数据库：** PostgreSQL 15+
- **ORM：** Prisma / TypeORM
- **认证：** JWT + OAuth2

### 基础设施：
- **云服务：** 阿里云 / 腾讯云
- **容器：** Docker + Docker Compose
- **编排：** Kubernetes（后期）
- **监控：** Prometheus + Grafana

## 📅 开发里程碑

### MVP阶段（12周）：
- **M1（第2周）：** 需求文档和设计稿完成
- **M2（第4周）：** 用户系统上线，内部测试
- **M3（第6周）：** 课程学习功能完成
- **M4（第8周）：** 能力认证系统完成
- **M5（第10周）：** 管理后台完成
- **M6（第12周）：** 正式上线

### 后续版本：
- **V1.1（上线后1个月）：** 学习社区、积分系统
- **V1.2（上线后3个月）：** 导师匹配、线下活动
- **V2.0（上线后6个月）：** 商城功能、高级认证

## 👥 团队架构

### 核心团队（4人）：
1. **产品经理/项目经理** - 需求管理、进度跟踪
2. **前端开发工程师（2人）** - 移动端+Web端开发
3. **后端开发工程师** - API开发、数据库设计
4. **UI/UX设计师** - 界面设计、用户体验

### 外部协作：
1. **医学专家（2-3人）** - 课程内容提供
2. **运营人员（1-2人）** - 内容审核、用户支持
